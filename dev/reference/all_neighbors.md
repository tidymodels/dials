# Parameter to determine which neighbors to use

Used in `themis::step_bsmote()`.

## Usage

``` r
all_neighbors(values = c(TRUE, FALSE))
```

## Arguments

- values:

  A vector of possible values (TRUE or FALSE).

## Examples

``` r
all_neighbors()
#> All Neighbors (qualitative)
#> 2 possible values include:
#> TRUE and FALSE
```
